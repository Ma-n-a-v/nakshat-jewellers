# Nakshat Jewellery – Headless Ecommerce Website

Nakshat Jewellery is a premium jewellery ecommerce website built with a **custom-coded frontend** and **Shopify-powered backend**.  
The project focuses on delivering a luxury user experience with clean UI, smooth animations, and a secure checkout flow.

This repository contains the complete source code for the website, including frontend components, Shopify integration, and deployment setup.

---

## Project Overview

The goal of this project is to create a modern jewellery website where:
- Customers can browse collections and products
- Add items to a cart
- Complete secure online payments
- Owners can manage orders, inventory, and shipping through Shopify

The frontend is fully custom-built based on a Framer design, while Shopify handles orders, payments, taxes, and fulfillment.

---

## Tech Stack

### Frontend
- Next.js (React)
- TypeScript
- Tailwind CSS
- Framer Motion (animations)

### Ecommerce & Backend
- Shopify (Products, Orders, Checkout, Payments)
- Shopify Storefront API (headless integration)

### Hosting & Deployment
- Vercel (recommended)
- Custom domain (connected via DNS)

---

## Key Features

- Custom luxury UI inspired by high-end jewellery brands
- Headless Shopify integration
- Secure Shopify-hosted checkout
- Product variants (size, material, etc.)
